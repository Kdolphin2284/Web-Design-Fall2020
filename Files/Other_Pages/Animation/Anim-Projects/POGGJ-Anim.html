I created animations of micro interactions on my redesign of OP.GG.

During my spring quarter at Drexel University, I took a micro interaction class in which we created animations for redesigned micro interactions of applications or websites that we chose to redesign.

Below are the five different micro interactions that I redesigned for the application and website called OP.GG.

First Interaction

This interaction shows how it would look if the user wanted to switch tabs from the homepage to the champions page.

Second Interaction

This interaction shows how it would look if the user clicks on the search bar to search for a specific champion.

Third Interaction

This interaction shows how the search results would look while the user is typing in a specific champion name.

Fourth Interaction

This interaction shows how the search page would switch to an individual champion page if the user clicks on a specific champion.

Fifth Interaction

This interaction shows how favoriting a champion looks.
